📖 Introduction

MOD STORE is an Android application designed to be your single, unified destination for discovering and managing applications across multiple platforms. Whether you're looking for PC software, Smart TV apps, or mobile applications, MOD STORE brings them all together in one beautifully crafted interface.

The app is free to download and does not require an account to browse. Connecting a GitHub account unlocks additional features like favorites sync, personalized library, and cross-device preferences.

ℹ️

What MOD STORE Is (and Isn't)

MOD STORE is a discovery and management client. It helps you find and track applications. All app installations are handled through each application's own installer or official source.

🚀 Getting Started

Follow these steps to get MOD STORE running on your Android device or Android-based Smart TV.

Installing on an Android Phone or Tablet

1

Download the APK

Visit the official download page (or GitHub Releases) and tap the latest .apk file to download it to your device.

2

Enable Unknown Sources

Go to Settings → Security → Install Unknown Apps (or Special App Access on newer Android). Allow your browser or file manager to install APK files.

3

Open the APK and Install

Tap the downloaded APK from your notification shade or file manager and follow the on-screen prompts to install MOD STORE.

4

Launch & Explore

Open MOD STORE. You can immediately browse PC, TV, and Android apps without signing in. To unlock the library and sync features, see the GitHub Authentication guide below.

💡

Minimum Requirements

MOD STORE requires Android 6.0 (Marshmallow) or higher. An active internet connection is needed to load app catalogs.

Installing on a Smart TV (Android TV / Google TV)

1

Transfer the APK

Download the APK on another device, then transfer it to your TV via a USB drive, or use a file-sharing app like Send Files to TV or Downloader to pull it directly.

2

Enable Unknown Sources on TV

Go to your TV's Settings → Device Preferences → Security & Restrictions and enable unknown sources for the Downloader or file manager app.

3

Install & Launch

Open the APK using your file manager and install it. MOD STORE will appear in your apps list and is optimized for TV navigation with a D-pad or remote control.

🔐 GitHub Authentication

MOD STORE uses GitHub OAuth to securely authenticate users. No passwords are ever stored — only a secure OAuth token is saved locally on your device to maintain your session.

ℹ️

Why GitHub?

GitHub authentication is used because MOD STORE is tightly integrated with the GitHub ecosystem. Your GitHub profile picture, username, and other metadata are used to personalize your experience inside the app.

Connecting Your GitHub Account

1

Open the Profile Tab

Tap the Profile icon in the bottom navigation bar (or swipe to the Profile screen). You will see a "Sign in with GitHub" button.

2

Authorize in Browser

Tapping the button opens GitHub's authorization page in your browser. Review the requested permissions — MOD STORE only requests your public profile access — and tap "Authorize".

3

Return to the App

After authorization, you will be redirected back to MOD STORE automatically. Your GitHub avatar and username will appear in the profile tab, confirming a successful login.

4

Sync Enabled

Features like the personal library, favorites, and synced preferences are now available. Your data is tied to your GitHub account for seamless cross-device access.

⚠️

Signing Out

To sign out, go to the Profile tab and tap "Sign Out". This will clear your session token from the device. Your favorites are preserved server-side and will re-sync when you sign back in.

🧭 Navigation Guide

Understanding the MOD STORE interface is simple. Here's a breakdown of the main UI elements.

Bottom Navigation Bar

The bottom navigation bar gives you quick access to the app's primary sections:

Tab Icon Purpose
Home 🏠 Main discovery feed showing all apps across all platforms.
PC Apps 🖥️ Filtered view showing only PC/Windows software.
TV Apps 📺 Filtered view showing only Smart TV / Android TV apps.
Library ❤️ Your personal collection of favorited apps (requires GitHub sign-in).
Profile 👤 GitHub account management, settings, and sign-in/out.

Category Chips

At the top of the Home and platform screens, you'll find horizontal scrollable category chips. Tap any chip to instantly filter the app list to that category (e.g., Games, Tools, Utilities, Media, Productivity). The currently selected chip is highlighted in Crimson Red.

App Detail Pages

Tapping any app card opens its full detail profile, which includes:

  • App icon, name, and developer information
  • A horizontal strip of scrollable screenshots
  • Version number, file size, and last-updated date
  • Full description and what's new / changelog
  • A prominent Install / Download action button
  • A heart icon to add the app to your Library

🔍 Search & Filter

MOD STORE's search is fast, intuitive, and works across all three platform types simultaneously.

Using Search

1

Tap the Search Bar

The search bar is prominently displayed at the top of the Home screen. Tap it to open the search keyboard.

2

Type Your Query

Results update in real-time as you type. Results are shown from all platforms (PC, TV, Android) unless you have a platform filter active.

3

Tap a Result

Tap any result to open the full app detail page. Use the back button to return to your search results.

Sorting Options

Tap the Sort button (usually a slider icon near the top-right of the app list) to choose a sort order:

Sort Option Description
LatestShows most recently added or updated apps first.
PopularShows apps with the highest engagement or download counts first.
AlphabeticalSorts apps A–Z by name.

❤️ Your Library

The Library is your personal collection of favorited apps. It's accessible from the bottom navigation bar and syncs automatically to your GitHub account.

💡

Requires GitHub Sign-In

The Library feature requires a connected GitHub account so your favorites can be synced across devices. See the GitHub Authentication guide to connect your account.

Adding Apps to Your Library

On any app detail page, tap the heart (♥) icon to add it to your Library. The icon will turn Crimson Red to confirm it's been saved. To remove an app, tap the heart icon again.

Viewing Your Library

Tap the Library tab in the bottom navigation to see all your saved apps in one place. Your Library is organized and searchable just like the main catalog.
